## 安装

- 基本用法
- 在git中下载代码后
```html
    1.安装node.js 因为依赖都是用node -v 6.0.0下载的。保证本地的node.js版本为6.0.0
    2.解压node_modules.zip到当前目录
```
```html
   //运行 develop
    npm run dev
```
```html
   //打包静态资源 build
    npm run build
```
